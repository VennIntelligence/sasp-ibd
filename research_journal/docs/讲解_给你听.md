# 我们在做什么 —— 写给你的大白话版

> 你说生信做得少、基本小白，所以这份文档我尽量不用术语，用比喻把"我们在干嘛、为什么、用了啥工具"讲清楚。看完你就能在汇报时自己讲出来。

---

## 一、一句话：我们想回答什么问题

**"细胞衰老"到底是不是 IBD（炎症性肠病）的"真凶"之一？还是只是跟着炎症一起出现的"路人"？**

这区别很重要：
- 如果是**真凶（因果）** → 抗衰老（清除衰老细胞）就可能成为治疗新思路；
- 如果只是**路人（相关）** → 抗衰老治疗就是白费劲。

我们第一阶段（转录组）发现：活动期 IBD 肠子里"衰老分泌信号(SASP)"确实升高、还能预测治疗效果。但**这只能说明"相关"，不能说明"因果"**——这正是你觉得"太浅"的地方。第二阶段就是来补"因果"这一刀。

---

## 二、为什么"相关"不等于"因果"（核心概念）

举个经典例子：夏天**冰淇淋卖得多**的时候，**溺水的人也多**。能说"吃冰淇淋导致溺水"吗？不能。背后真正的原因是"天热"——天热既让人买冰淇淋，也让人去游泳。冰淇淋和溺水只是"被同一个原因带着一起涨"，是**相关**，不是**因果**。

放到我们这：IBD 肠子里"衰老信号高"和"病情重"同时出现。可能是：
- (A) 衰老 → 导致 → IBD（真凶）；
- (B) IBD 的炎症 → 顺便造成 → 衰老信号高（路人/结果）；
- (C) 有第三个因素同时推高两者。

光看表达谱（第一阶段）分不清这三种。怎么办？

---

## 三、怎么判断因果：孟德尔随机化(MR) —— "老天爷帮我们做的随机对照试验"

**金标准是随机对照试验(RCT)**：把人随机分两组，一组"衰老高"、一组"衰老低"，看谁更容易得 IBD。但你不可能真的去"随机分配人的衰老水平"——不现实也不道德。

**孟德尔随机化(MR)** 的妙处：**老天爷在你出生那一刻，已经帮你"随机掷骰子"分配了基因。** 有些人天生带的某个基因变异(SNP)会让某个衰老基因表达得高一点，有些人低一点——这个分配是**受精时随机发生的**，跟你后天的饮食、炎症、环境无关。

于是我们可以这样推：
> 找一个"天生就让某衰老基因表达更高"的基因变异 → 看带这个变异的人是不是更容易得 IBD。
>
> 如果是 → 说明"衰老基因表达高"**因果地**增加 IBD 风险（因为基因是出生就定的、不可能被 IBD 反过来影响，时间顺序铁定是"基因在前、病在后"）。

这就把"相关"升级成了"因果"。这就是你提到的**孟德尔随机化**。

**用大白话总结 MR 三个前提**（汇报时可讲）：
1. 这个基因变异**真的**影响该衰老基因的表达（强相关，我们要求很显著）；
2. 这个变异**只**通过"该基因"影响 IBD，不走别的捷径（这是难点，靠下面的 coloc 等来加固）；
3. 这个变异不跟"混杂因素"绑定（因为是随机分配的，基本满足）。

---

## 四、为什么还要"共定位(colocalization)" —— "是不是同一个真凶"

MR 有个软肋：基因组上挨得近的变异会"连锁"（手拉手一起遗传）。可能是：**变异 X 影响衰老基因，旁边的变异 Y 影响 IBD，X 和 Y 手拉手**——这样 MR 会被骗，以为是衰老基因导致 IBD，其实是隔壁的 Y 在搞事。

**共定位**就是来排查这个的："影响衰老基因的那个变异" 和 "影响 IBD 的那个变异"，**到底是不是同一个？**
- 是同一个(PP.H4 高) → 真·因果，可信；
- 是两个挨着的邻居(PP.H3 高) → 假象，剔除。

比喻：两起案子，一个嫌疑人。共定位就是确认"两案是不是同一个凶手干的"，而不是"两个长得像的邻居各干一件"。

## 五、我们具体用了什么数据和工具（汇报时报得出来）

| 环节 | 用了什么 | 通俗解释 |
|---|---|---|
| 衰老基因清单 | **SenMayo**（124 个衰老/SASP 基因） | 公认的"细胞衰老"基因名单 |
| "基因→表达"的随机骰子 | **eQTLGen**（3.1 万人血液 eQTL） | 找"天生让某基因表达高/低"的变异 |
| "变异→IBD"的证据 | **de Lange 2017 IBD GWAS**（2.5 万病例） | 看这些变异和 IBD 的关系 |
| 因果推断 | **孟德尔随机化(MR, Wald ratio)** | 老天爷的随机试验 |
| 排查假象 | **共定位(coloc, ABF 方法)** | 是不是同一个真凶 |
| 第一阶段佐证 | GEO 肠黏膜表达谱 + 单样本评分 | 病人肠子里这些基因真的高、且随治疗变化 |

全部是公开数据 + 我用 Python 自己实现的算法（你电脑没装 R，我用 Python 全程搞定）。

## 六、结果（核心发现）

我们对 82 个有"遗传骰子"的衰老基因，逐个做了 MR（看它们是否因果影响 IBD），多重检验校正后：

**IBD：9 个衰老基因被判定为"因果"（FDR<0.05）**，而且方向有正有反——这是很有意思的深层发现：

| 基因 | 方向 | 通俗含义 |
|---|---|---|
| **TNFRSF1A**(TNF受体) | 风险(OR≈1.16) | **教科书级验证**：它正是"抗 TNF 药"(类克等)的靶点，MR 把它揪出来，说明我们方法可信 |
| CCL8 / PTBP1 / HGF / ICAM1 / PLAUR | 风险(OR>1) | 这些衰老/SASP 基因表达越高，IBD 风险越大——潜在"坏分子"/治疗靶点 |
| **CXCR2 / MMP9 / IL1B** | 保护(OR<1) | 反直觉：它们表达高反而 IBD 风险**低**——说明"衰老信号"不全是坏的，有的可能是保护性修复 |

**CD（克罗恩）7 个、UC（溃结）5 个**也各有因果基因，CXCR2、MMP9、TNFRSF1A 在多个亚型反复出现，稳健。

**最关键的两点（汇报亮点）：**
1. **从"相关"升级到"因果"**：我们不再只是说"衰老和 IBD 相关"，而是能指名道姓"这几个衰老基因因果驱动/保护 IBD"。
2. **方向有正有反**——纠正了"衰老 = 一律有害"的简单认知，这比"又一个 signature"深刻得多。
3. **直接连到药**：揪出的因果基因里，TNFRSF1A(抗TNF)、IL1B(抗IL-1)、MMP9(andecaliximab)、ICAM1(alicaforsen)、CXCR2(拮抗剂) **都已有对应药物或在研**——说明这些因果基因是**可成药**的，临床落点很实。

**共定位(coloc)进一步把关——这是最硬的一步：**

MR 找出 9 个"嫌疑人"，但 coloc 一查"是不是同一个真凶"，发现大多数是**连锁假象**（影响衰老基因的变异和影响 IBD 的变异只是"邻居"，不是同一个）。最后**只有 2 个基因真正过关**（PP.H4>0.9，即"同一个真凶"证据充分）：

| 基因 | 因果方向 | 共定位 PP4 | 肠黏膜表现 | 一句话 |
|---|---|---|---|---|
| **CCL8**(单核细胞趋化因子) | **风险** OR≈2.35 | **0.955** ✅ | 活动期↑ | 因果"坏分子"，可作靶点 |
| **CXCR2**(中性粒细胞受体) | **保护** OR≈0.75 | **0.951** ✅ | 活动期↑(代偿) | 反直觉亮点（见下） |

> 其余像 ICAM1、FGF2、PTBP1 等，MR 看着显著，但 coloc 显示 PP.H3≈1（是"两个邻居"不是"同一个真凶"）→ **被正确剔除**。这说明我们的流程严谨，不是抓到啥都信。

**CXCR2 这个发现特别有意思（汇报可重点讲）：**
- 它在**遗传上是保护**的（天生表达高 → IBD 风险低），但在**发炎的肠子里却升高**——说明它在病里的升高更像是"身体的代偿/修复反应"，而不是致病。
- 它有现成的**拮抗剂(抑制剂)药物**。但我们的结果提示：**盲目抑制 CXCR2 可能反而有害**（因为它是保护性的）——这是个有临床价值的"反向提醒"，是单纯做 signature 绝对得不出的洞见。

**因果方向有正有反**，再次说明"衰老/SASP 不是一律有害"——这是整篇最有深度、最不烂大街的地方。

## 七、这套设计的诚实局限（汇报主动讲，显严谨）

- MR 假设之一"只通过该基因起作用"无法 100% 证明，coloc 只能加固不能终证；
- eQTLGen 是**血液**的 eQTL，肠道特异性有限（我们用结肠 eQTL 做补充）；
- 很多"诱导型"SASP 因子（IL6、IL8 等）在健康组织里不表达，找不到基因工具——这类只能靠表达谱/未来蛋白 QTL 补；
- 因果方向我们也可做"反向 MR"（IBD→衰老）来对照。

---

## 八、为什么这比第一版"高级"、且不烂大街

第一版只是"衰老信号和 IBD 相关 + 能预测"——描述性，审稿人一句"只是相关"就压住。
加上 **MR+共定位** 后，我们能说**"某些衰老基因是 IBD 的因果驱动者"**——这是**因果级**证据，且"系统性地对整个衰老基因集做 MR+coloc 筛因果基因"在 IBD×衰老领域还没人做过，既新又不灌水。
