## 一、为什么转（证据，非拍脑袋）

2026-06-23 做了三路第一性原理探索（benchmark / 无偏 meta-DE / 文献定位），结论高度收敛：

1. **「SASP 预测应答」并不特殊——它就是「炎症」的别名。**
   - 池化 AUC：SASP 0.80，泛炎症评分 0.79（持平/略高）；OSM、IL13RA2、TREM1、中性粒signature 全挤在 **0.80–0.82** 天花板。
   - SASP 与泛炎症 r=**0.88**，加进炎症模型**零增量**（LRT p≈0.10）；无泄漏数据驱动signature 也只到 0.80。
   - `inflammation_adjusted` 显示 SASP 只赢**内镜 Mayo**(弱对手，AUC 0.28)，赢不了**分子炎症**（增量≈0）。

2. **无偏分析：难治的主导生物学不是衰老。**
   - 难治(NR)上调 = **中性粒(OR 154) > OSM-炎症成纤维(OR 105) > TNF/NFκB > 单核 > 干扰素**；下调 = 上皮分化丢失(CDX2/NR3C2↓)。
   - **衰老停滞臂(p16/p21/CDKN)彻底为零(p=0.22)**；SASP 排第 2 纯属**派生**——只是中性粒/OSM 共享的趋化-蛋白酶基因换了「衰老」标签。

3. **这片地早被占了；唯一没人做的是因果遗传。**
   - OSM(West 2017)、TREM1、GIMATS(Martin)、UC1/UC2(Czarnewski)、NET signature 已反复发表同一炎症-基质生物学。「SASP 预测应答」= 重新发现已知。
   - **差异化 = 对应答相关基因做系统 MR+coloc**——现有所有 IBD 应答signature 论文都没有。

**结论**：senescence×IBD 牵强且弱；response-prediction 不够新。项目真正稀缺的资产是**那台 MR+coloc 因果引擎**，但它一直瞄错了基因集（82 个「衰老基因」）。

## 二、新论点（New Thesis）

把因果引擎对准**数据与文献真正指认的「难治性 IBD 炎症-分泌模块」**（中性粒 + OSM-炎症成纤维 + 单核/趋化-蛋白酶），回答一个**没人答过**的问题：

> **「难治性 IBD 公认的那些炎症标志物（OSM/OSMR/TREM1/IL13RA2/CXCR2/CCL8/中性粒基因…）里，哪些是因果驱动者、哪些只是路人？」**

并且用**肠道/结肠 eQTL（GTEx colon）而非只用血液 eQTLGen**——血液 eQTL 做肠病是审稿人必打的点。

**头条 = 因果地图**（含方向 + 可成药性 + 反直觉提醒，如 CXCR2 遗传保护却是药靶 → 慎用拮抗剂）。
**支撑 = 描述层**：bulk 多队列跨药类预测（SASP 诚实降为「炎症模块的一个读数，AUC≈0.8 饱和家族成员」）+ 单细胞定位（模块由哪类细胞驱动）。

## 三、各层在新框架下的定位

| 旧定位 | 新定位 |
|---|---|
| 「衰老时钟」阴性 | 保留为诚实阴性对照（与 LTL 阴性呼应） |
| 「SASP 预测应答」(原卖点) | **降级**为描述性支撑：炎症模块的饱和预测器之一，诚实 benchmark 进 OSM/TREM1/中性粒家族 |
| 「SASP vs 停滞解耦」 | 保留为「该模块是分泌-炎症而非细胞衰老」的机制澄清 |
| MR+coloc 筛 82 衰老基因 → CCL8/CXCR2 | **升级为头条**：MR+coloc 筛**难治炎症模块**基因（+肠道 eQTL）→ 因果地图。CCL8（因果风险, coloc 0.955, FinnGen 复制 OR 4.37）是现有最硬锚点 |
| 单细胞「找真衰老细胞」+ 基础模型扰动 | 扰动阴性降为方法注脚；单细胞**转做模块的细胞来源反卷积** |

**现有最硬的两个遗传锚点（继续用）**：CCL8（MR 2.35 / coloc 0.955 / FinnGen IBD 复制 OR 4.37）、CXCR2（MR 0.75 / coloc 0.951 / 方向性复制；保护却可成药 = 临床反直觉提醒）。
